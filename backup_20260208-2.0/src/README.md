# Mine Shop - Accountability-First Shopping Platform

A React + Firebase shopping platform that solves the "mine" problem with user accountability and time-limited reservations.

## Features

- ✅ User authentication required
- ✅ 48-hour automatic reservation expiration
- ✅ Admin panel for product management
- ✅ Real-time stock tracking
- ✅ Multiple stock support per product
- ✅ User accountability tracking
